# Live Polling System

Real-time classroom polling system where teachers create polls and students vote live.

## Architecture

Frontend:
React + TypeScript + Bootstrap
Deployed on Vercel

Backend:
Node.js + Express + Socket.io
Prisma ORM + Supabase PostgreSQL
Deployed on Render/Railway

## Features

Real-time polling
Live results
Timer synchronization
State recovery
Poll history
